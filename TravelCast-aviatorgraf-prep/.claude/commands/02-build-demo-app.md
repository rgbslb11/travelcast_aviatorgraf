# 02-build-demo-app — TravelCast Claude Code Command

Build or repair the demo-mode TravelCast AviatorGraf Prep app.

Requirements:

- Plain HTML/CSS/vanilla JS.
- No framework.
- No real API calls.
- No secrets.
- Demo data must include 10 airports.
- DEN must include GDP/thunderstorm example.
- Airport Status Board must render.
- Airport Detail must render.
- Export buttons must work.

After building, run a self-audit and fix issues.
